Documentación de API - Feriados

GET /holidays

Devuelve todos los feriados disponibles o filtra por país y rango de fechas.

GET https://www.feriadito.com/holidays
GET https://www.feriadito.com/holidays?countryCode=AR&from=2025-05-01&to=2025-04-29 
[
  {
    "Id": 6,
    "Name": "Dia del Trabajador",
    "Day": "2025-05-01T00:00:00Z",
    "CountryId": 1,
    "CountryCode": "AR"
  }
]

GET /holidays/:id

Devuelve un feriado específico por ID.

GET https://www.feriadito.com/holidays/1

GET /holidays/verify

Verifica si una fecha es feriado en un país determinado. Retorna un booleano.

GET https://www.feriadito.com/holidays/verify?countryCode=AR&date=2025-05-01 
{
  "exist": true
}

GET /countries

Devuelve todos los países registrados.

GET https://www.feriadito.com/countries 
[
  { "Id": 1, "Name": "Argentina", "CountryCode": "AR" },
  { "Id": 2, "Name": "United States", "CountryCode": "US" },
  { "Id": 3, "Name": "Mexico", "CountryCode": "MX" },
  { "Id": 4, "Name": "Colombia", "CountryCode": "CO" }
]

GET /countries/:id

Devuelve un país por ID.

GET https://www.feriadito.com/countries/1 
{
  "Id": 1,
  "Name": "Argentina",
  "CountryCode": "AR"
}
